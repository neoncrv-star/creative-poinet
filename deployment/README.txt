نظام النشر لمشروع Creative Poinet
================================

هذا المجلد يحتوي على كافة الأدوات اللازمة لنشر المشروع على سيرفر الإنتاج (Hostinger).

الملفات المتاحة:
1. publish_all.bat (موصى به):
   يقوم برفع التعديلات إلى GitHub ثم الدخول للسيرفر عبر SSH وتحديث الكود تلقائياً. هذا هو الخيار الأسرع والآمن لنشر "التغييرات فقط".

2. 1-initial_setup.bat:
   يستخدم لمرة واحدة فقط عند تهيئة المستودع وربطه بـ GitHub.

3. 2-push_to_github.bat:
   يرفع التعديلات إلى GitHub فقط بدون تحديث السيرفر.

4. 3-update_server.bat:
   يحدث السيرفر من GitHub (يفترض أنك رفعت التعديلات مسبقاً).

5. manual_scp_deploy (ps1/sh):
   ملفات للنشر اليدوي الكامل عبر SCP (لا ينصح بها إلا في حالات الضرورة).

ملاحظة: تأكد من إعداد مفتاح SSH للوصول إلى سيرفر Hostinger بدون كلمة مرور.
